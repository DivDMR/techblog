CSS BEM

BEM is Block Element Modifier and a way to create consistent CSS classes. Using BEM you give descriptive classes of (almost) every single HTML element that you have, and it allows you to change everything individually without having confusing CSS

See the Pen BEM Example by DivDMR (@divdmr) on CodePen.

CSS Grid

CSS Grid Layout grid-based layout system that completely changes the way that you can design user interfaces.

CSS Grid is described right in its name, it is a layout system that allows you to layout the HTML elements of your page in a grid-like manner
You can set columns and rows and give them height and width and you have a large degree of choices on how to position the elements instead of the grid.
