AddEventListener

Here we were introduced to the "AddEventListener" method in javascript. In javascript there are many events that the browser will track while the site is running and for this program we utilized the "click" event. This event listens for whenever the user inputs a click on whatever item was defined within the code. Additionally after detecting the event the program will run the function specified.

On top of that we learned how to access classes using javascript with keywords like ".contains", ".add", ".toggle", ."remove". These keywords allowed us to add different colored backgrounds (through classes) whenever we clicked the increase and decrease button.

Color & Shape Sliders

Here we learned how to use user input to modify the shape and color of a box (just a div). This was accomplished through the use of "range" inputs which created sliders that could be moved by the user. In the javascript we once again used the AddEventListener method call a function whenever you moved the sliders
