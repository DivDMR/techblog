Two-Sided Tags

See the Pen Two Sided Tages by DivDMR (@divdmr) on CodePen.

Two-sided tags have an open and close tag. The close tag has a “/” before the element. You can code tags inside of other tags.

One-sided tags

See the Pen Untitled by DivDMR (@divdmr) on CodePen.

One-sided tags go do not have a closing tag. You can include the "/" at the end of the tag

Attributes

See the Pen Attributes by DivDMR (@divdmr) on CodePen.

Attributes provide more information about a tag. They have the attribute name followed by an equal sign along with a value in quotes

Semantic Tags

See the Pen Semantic Tags by DivDMR (@divdmr) on CodePen.

A semantic tag is descriptive and describes the content. These semantic group tags can be nested in a variety of ways. Whenever using a group tag, ask yourself, “Is there a semantic tag I could use instead?”
